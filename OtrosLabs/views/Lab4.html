Preguntas:

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    2. En cuanto a semejanzas, tienen un nombre similar, proceden de C y tienen una sintaxis similar.
    En cuanto a diferencias, java es un lenguaje de programación orientado a objetos puros, mientras que javascript esta basado en prototipos, java es un lenguaje compilado y javascript interpretado, java presenta un tipado fuerte, pero javascript tiene un tipado débil, java se ejecuta en la maquina virtual java y javascript se ejecuta en el navegador y finalmente java es un lenguaje de programación tipo estatico, mientras javascript es de tipo dinamico.

  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

    3. El objeto Date tiene muchos métodos, algunos de estos son Date.now(), Date.parse(), Date.UTC(), Date.prototype.getDate(), Date.prototype.getDay(), Date.prototype.getFullYear(), entre otros.

  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    4. Los arreglos tambien poseen una gran variedad de métodos, como push(), unshift(), pop(), shift(), slice(), concat(), etc.

  4. ¿Cómo se declara una variable con alcance local dentro de una función?

    5. Declarando la variables como var, let o const dentro de una función, pero en el caso de let y const estas son variables locales con ámbito de bloque.

  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    6. Que estas variables podrian interferir en otras funciones si hay varias con el mismo nombre, por lo que siempre se deben declarar las variables pensando en el alcance que se requiere que tengan.
