Preguntas:

  1. ¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?

    2. Debido a que de esta manera uno se asegura del correcto funcionamiento del servidor con las formas obtenidas, ademas de evitar que usuarios maliciosos busquen enviar datos erroneos al servidor saltando la validación de la pagina html.

  2. ¿Cómo puedes saltarte la seguridad de validaciones hechas con JavaScript?

    3. Simplemente deshabilitando el javascript en el navegador.

  3. Si te puedes saltar la seguridad de las validaciones de JavaScript, entonces ¿por qué la primera pregunta dice que es una buena práctica?

    4. Debido a que permite una mejor experiencia al usuario y evita los retrasos de la comunicación entre el cliente y servidor.
